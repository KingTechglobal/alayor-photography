# Alayor Photography

Premium photography studio website for **Alayor Photography** — Studio sessions for Birthdays, Maternity, and Preweddings in Lagos, Nigeria.

## 📸 Live Preview

Open `index.html` in your browser to view the website.

## 🛠️ Tech Stack

- **HTML5** — Semantic markup
- **CSS3** — Glassmorphism, animations, responsive design
- **JavaScript** — Interactive carousel, portfolio filtering, 3D tilt effects
- **Google Fonts** — Outfit typeface

## 📁 Project Structure

```
├── index.html    # Main HTML page
├── style.css     # All styles and responsive breakpoints
├── main.js       # Interactive features and animations
├── logo.png      # Alayor Photography logo
└── ceo.jpg       # CEO & Lead Photographer photo
```

## ✨ Features

- 🎭 **Glassmorphism UI** with dark theme and red accent gradients
- 💰 **Pricing cards** with 3D tilt hover effects
- 🖼️ **Portfolio gallery** with category filtering
- 💬 **Testimonial carousel** with Google Reviews
- 📱 **Fully responsive** mobile-first design
- ✨ **Scroll reveal animations** throughout

## 📝 Note

Binary assets (`logo.png` and `ceo.jpg`) need to be added manually to the repository.
